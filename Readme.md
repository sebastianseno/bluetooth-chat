# Bluetooth Low Energy Chat App
Chat with bluetooth connection


## Library

- Dagger hilt
- Jetpack compose
- Material
- Accompanist

## How to Use Apps


- Open Application.
- Click scan now button to scan nearby devices.
- Click connect now button to pair with another device. 
- Pair will automatically and no need pass key
